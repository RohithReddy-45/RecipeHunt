# Recipehunt

## What is Recipehunt?

Recipehunt is a recipe web application,where a user can search for a variety of recipes and find the ingredients and quantities needed to make each recipe item.

It provides a simple and intuitive interface for finding new and delicious recipes.

## Live Site

Check out the live site [RecipeHunt](https://recipehunt-rohith.netlify.app/).

## Technologies used

HTML, Tailwind CSS, and JavaScript.

## How to use Recipehunt?

Download or clone this repository and run the following command in terminal:

```
npm install
or
npm i
```
